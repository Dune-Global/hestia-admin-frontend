---

## How to run project locally ?

1. Clone the repository

```bash
git clone https://github.com/Dune-Global/hestia-admin-frontend.git
```

2. Change directories

```bash
cd hestia-admin-frontend
```

3. Install dependencies

```bash
yarn
```

4. Start the server

```bash
yarn dev
```
